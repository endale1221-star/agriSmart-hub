 ## Pages & Features

1. Homepage (`index.html`)
  - Eye-catching hero section with “Welcome to AgriSmart Hub” .
  - Features section showcasing:
  - Smart Farming Technology
  - Market Access Platform
  - Climate Advisory Services
   - Fully responsive Bootstrap 5 cards.
   - Footer with social media icons and copyright.

## 2. About Page (`about.html`)
- Two-column layout with Mission.png and Vision.png.
- Mission & Vision cards highlighting platform goals.
- Progress bars showing key achievements:
  - Farmer Digitalization: 80%
  - Smart Market Adoption: 65%
  - Climate-Ready Farms: 70%

## 3. Services Page (`services.html`)
-  6 Bootstrap cards displaying core services:
  - Smart Irrigation
  - Drone Mapping
  - Digital Marketplace
  - Soil Analysis
  - Farm Data Analytics
  - Crop Monitoring & Advisory
- Includes images and  “Learn More”  buttons for each service.

## 4. Innovations Page (`innovations.html`)
- Responsive carousel with 3 innovation images (IN1.png, IN2.png, IN3.png) and captions.
- Embedded YouTube video demonstrating smart farming technologies.
- Clean, professional design with rounded images and readable captions.

## 5. Contact Page (`contact.html`)
- Bootstrap contact form: Name, Email, Subject, Message.
- Sidebar card with contact details (phone, email, office address).
- Embedded Google Map iframe showing office location.

## Technologies Used
- HTML5 – Semantic and structured markup.
- Bootstrap 5 (via CDN) – Responsive grid, components, and utilities.
- CSS (style.css) – Custom styling, hover effects, color scheme, and typography.
- Images – Local folder for all project images to ensure full offline functionality.

## Design & Styling
- Color Scheme: Soft greens (#2e7d32, #81c784) with light background (#f1f8e9).
- Typography: Clean, readable fonts with headings bold and distinct.
- Responsiveness: Fully mobile-first design, working seamlessly on all devices.
- Visual Enhancements: Hover effects on cards and buttons, rounded corners, smooth transitions.

## Key Features
- Fully functional local navigation across 5 pages.
- Interactive carousel and embedded video on Innovations page.
- Engaging, readable hero section on homepage.
- Progress tracking visuals on About page for enhanced presentation.
- Professionally structured CSS and folder hierarchy for maintainability.
